Obsidian RAG Protocol — Data Flow

How two AI agents share long-term memory through a single vault index

Zero-Token Context Injection

  • • Index is read as a tool call (~15KB), not prompt-prefixed
  • • Agent fetches only matched notes via read_file
  • • No context window pollution from irrelevant notes

Session-Start Awareness

  • • Byte-offset digest into wiki/log.md (v1.4)
  • • Each agent knows what the other wrote since last session
  • • Best-effort — never blocks agent startup

Bidirectional Collaboration

  • • CC writes wiki/ · Hermes writes hermes-knowledge/
  • • log.md serves as append-only communication channel
  • • Hash-based incremental rebuilds — no mtime drift
